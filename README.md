# MotionCore Studio

A premium, futuristic website for MotionCore Studio - a digital agency specializing in high-end web design and social media presence.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Dark Mode Only**

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3001](http://localhost:3001) in your browser.

## Deployment

This project is ready to deploy to Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy

The site will be automatically optimized and deployed.

## Project Structure

- `/app` - Next.js App Router pages
- `/components` - React components
- `/components/sections` - Page sections (Hero, Pricing, etc.)

## Design Philosophy

- Futuristic and cinematic
- Premium and authoritative
- High-contrast dark aesthetic
- Subtle, intentional animations
- No playful or bubbly design elements
